About API Keys

API keys allow users to authenticate from the CLI. Each key is tied to a specific host machine.

  • Users request keys via pulldb register or pulldb request-host-key
  • New keys require admin approval before they can be used
  • Revoking a key immediately blocks CLI access from that host
  • Users can request multiple keys for different machines
