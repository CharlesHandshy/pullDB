Web UI Overview

The pullDB Web UI provides a modern, browser-based interface for managing database restores. Submit jobs, monitor progress, manage API keys, and administer the system—all from your browser.

Overview

The Web UI offers the same functionality as the CLI but with a visual interface that makes it easy to monitor multiple jobs, browse backup history, and manage your account settings.

Dashboard

At-a-glance view of your jobs, system status, and quick actions.

Restore Wizard

Step-by-step guided workflow for submitting restore jobs.

Job Monitor

Real-time progress tracking with detailed phase information.

Admin Panel

User management, host configuration, and system settings.

Accessing the Web UI

The Web UI is served on port 8000 by default. Access it via your browser:

Browser URL
http://your-server:8000

Authentication

The Web UI supports two authentication methods:

Username & Password

Log in with your pullDB username and password. This creates a session cookie valid for 24 hours.

API Key

Use an API key in the Authorization header for programmatic access or when integrating with other tools.

First-time login?
Use the admin credentials provided during installation. You should change your password immediately after first login.

User Roles

pullDB has three user roles with different permission levels. Your role determines what you can see and do in the UI.

ADMIN 👑

Full system access. Can manage users, hosts, API keys, and system settings. Sees all jobs from all users.

Access: Everything
MANAGER 📋

Can view all jobs across the system and access the Manager view. Cannot modify system settings or manage users.

Access: Dashboard, Jobs, Manager View, Profile
USER 👤

Standard user access. Can submit restore jobs, view their own jobs, and manage their profile and API keys.

Access: Dashboard, Jobs (own), Restore, Profile

Navigation

The Web UI uses a sidebar navigation pattern. The main navigation items change based on your role.

Menu Item Description Roles
Dashboard Home page with job summary and quick actions All
Restore Submit new restore jobs All
Jobs View and manage restore jobs All
Manager Overview of all system jobs Manager, Admin
Admin System administration panel Admin only
Profile Account settings and API keys All

Error Handling

The Web UI displays user-friendly error pages when something goes wrong. Each error type has a dedicated page explaining the issue and possible solutions.

Table Components

Data tables throughout the UI share common features for filtering, sorting, and navigation. Understanding these components will help you work efficiently with lists of jobs, users, and hosts.

Theme Settings

The Web UI supports both light and dark themes. Toggle between them using the theme button in the navigation bar, or use the keyboard shortcut.

Ctrl + Shift + L Toggle light/dark theme

Your theme preference is saved in your browser's local storage and persists across sessions. The UI also respects your operating system's color scheme preference on first visit.
